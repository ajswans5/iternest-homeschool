# IterNest

IterNest is a homeschool workflow application for planning, organizing, and managing learning across a family.

## Project Structure

- `app/` contains the production React + TypeScript + Vite application.
- `docs/` contains product notes, feature specs, roadmaps, decisions, and planning documents.
- `assets/` contains repo-level brand and marketing assets such as logos, icons, and reference images.
- `prototypes/` contains exploratory HTML or UI prototypes that are not production app code.
- `scripts/` contains utility scripts for project maintenance and developer workflows.

## Development

The application lives in `app/`.

```bash
cd app
npm install
npm run dev
```
