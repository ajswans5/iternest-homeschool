# Roadmap

This roadmap is governed by `constitution.md`. Future work should preserve trust sequencing: source evidence before interpretation, interpretation before recommendation, and parent approval before meaningful changes.

## Current Stage

IterNest is currently at the Trust Foundation + Single Lesson Understanding Prototype stage.

The product is not ready for a full end-to-end MVP path test yet. The next milestone is a prerequisite proof milestone, not a broader vertical slice.

## Next Milestone: Single Real Lesson Reliability

Goal: make IterNest reliably analyze one complete real lesson from one real uploaded curriculum PDF.

This milestone must come before:

- Scheduling
- Plan repair
- Calendar logic
- Learner profiles
- Curriculum-wide import
- Full end-to-end MVP testing

Acceptance criteria:

1. Select one real lesson from the PDF.
2. Avoid showing a confusing raw text dump as the main user result.
3. Clearly identify what the parent teaches.
4. Clearly identify what the student does.
5. Identify materials, review, time, dependencies, and flexibility only when supported by source evidence.
6. Mark unclear or unsupported items as Needs Parent Review.
7. Attach source evidence to every conclusion.
8. Allow the parent to confirm: "Yes, that is what this lesson is asking me to teach."

## Later Milestones

- Repeat reliable lesson understanding across multiple lessons.
- Build curriculum-wide understanding after single-lesson reliability is proven.
- Add learner profiles only after lesson understanding is trustworthy.
- Add schedule/calendar integration only after IterNest understands lesson intent and flexibility.
- Add plan repair after curriculum understanding, lesson intent, learner needs, and flexibility are all represented clearly.

## FounderFlow Refinement

When a project has not yet reached a vertical-slice stage, FounderFlow should identify the prerequisite proof milestone instead of asking for a full end-to-end path test.

For IterNest, that prerequisite proof milestone is Single Real Lesson Reliability.
